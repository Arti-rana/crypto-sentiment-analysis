# Bitcoin Market Sentiment vs Trader Performance Analysis

## 📌 Objective
To analyze how market sentiment (Fear-Greed Index) impacts trader performance using historical trading data.

---

## 📂 Dataset
- Historical trading data (Hyperliquid)
- Bitcoin Fear-Greed Index dataset

---

## ⚙️ Methodology
- Cleaned and preprocessed both datasets
- Extracted timestamp and handled missing values
- Created key metrics such as Profit & Loss (PnL)
- Performed segmentation based on trade size
- Analyzed performance across sentiment categories
- Visualized results using boxplots

---

## 📊 Key Insights
- Trader performance is similar across sentiment categories
- Extreme market conditions show higher volatility
- Most trades result in small profits/losses
- High-value trades carry higher risk and reward
- Market sentiment impacts volatility more than profitability

---

## 📈 Strategy Recommendations
- Reduce exposure during Extreme Fear conditions
- Avoid overtrading during Greed phases
- Use proper risk management (stop-loss, position sizing)
- Be cautious with large trade sizes
